# biotab: the Swiss army knife of tabular data in bioinformatics

> Under development

## Subcommand

* xlsx2tab
* tab2xlsx
* join
* venn
* upset
* upsetbed


## Examples


```
./biotab xslx2tab -i test.xlsx -p test
```


```
./biotab tab2xlsx -t test.si-data.txt,test.sj-data.txt -s a,b
```
